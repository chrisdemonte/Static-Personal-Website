Work Experiences

Software Developer, Intern

Brookhaven National Laboratory | June 2022 - Present

Collaborated with scientists at the IOS Beamline, National Synchrotron Light Source II, Brookhaven National Laboratory, to develop software that improves the efficiency and user experience of data acquisition, processing, and analysis in X-ray photoelectron spectroscopy.

  • Created a user interface in the Igor Pro programming language that provides tools for processing XPS data, including reading raw data, preprocessing, file managment/organization, normalization, calibration, and visualization.
  • Created a user interface in the Igor Pro programming language for analyzing XPS data through statistical curve fitting, and provides tools for visualizing curves, multi-curve fitting, and analysis on large batches of data.
  • Developed user interfaces with Python and PyQt that interface with the NSLS-II Bluesky control system to scan samples at the IOS beamline, schedule experimental procedures, and visualize collected data.
  • Technologies Used: Igor Pro 9, Python, Qt/PyQt, Qt Designer, iPython, BlueSky, Linux.

Software Developer

Adelphi University | October 2019 - January 2021

Developed software that is currently used by the Adelphi University Psychology Department to recreate an experiment that investigates the theory of depressive realism. In the original 1970s experiment, subjects would press a button that had either a positive or negative affect on a lightbulb turning on.

  • Conducts the experiment through a Java based user interface. Upon completion of the experiment, the program runs statistical analysis on the data and outputs the results in a text file.
  • Later adapted to JavaScript so the experiment could be hosted on the Qualtrics Survey website. Utilizes the Qualtrics Survey Engine for data collection, analysis, and storage, with custom JavaScript for managing survey flow.
  • Submitted to the 2019 SURC Undergraduate Conference. Research paper currently pending.
  • Technologies Used: Java 8, JavaFX, JavaScript, Qualtrics.
